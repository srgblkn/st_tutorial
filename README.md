# Streamlit-приложение (3 страницы)

1. Рынок акций.py — котировки Apple/Tesla/MSFT/Google через yfinance, графики цены и объёма  
2. Замена пропусков в CSV-файлах.py — загрузка CSV, поиск пропусков, автозаполнение (медиана/мода), скачивание результата  
3. Чаевые.py — загрузка tips.csv, фильтр по дням, 3 графика (сумма по дням, пол, scatter), скачивание CSV и PNG  

#### Все страницы на русском, виджеты в sidebar, есть st.file_uploader + st.download_button  
#### Используются st.line_chart, st.bar_chart, matplotlib
